## 🧠 SYSTEM ARCHITECTURE
───────────────────────────────────────────────────────────────────────

 INPUT: Web Server Logs
    ↓
 PHASE 1: RECONNAISSANCE (Passive + Active Analysis)
    ├─ Passive Agent: Parse logs, detect patterns
    └─ Active Agent: Server fingerprinting, endpoint discovery
       ↓ (Context passed via CrewAI Memory)
 PHASE 2: VULNERABILITY ASSESSMENT (Threat Detection & Scoring)
    ├─ Threat Detector: Detect attacks (SQLi, XSS, DDoS, etc.)
    └─ Classifier: CVSS scoring & severity assignment
       ↓ (Context passed via CrewAI Memory)
 PHASE 3: REPORTING & ALERTS (Generate Reports & Alerts)
    ├─ Report Generator: Professional security report
    └─ Alert Manager: Real-time security alerts
       ↓
 OUTPUT: JSON Report + Security Alerts
